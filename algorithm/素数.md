# 素数

筛选法

```java
static List<Integer> get(int n){
    int[] nums = new int[n];
    List<Integer> p = new ArrayList<>();
    p.add(2);
    for(int i=3;i<=n;i+=2){
        if(nums[i]==0)
            p.add(i);
        for(int m : p){
            if(i*m>n) break;
            nums[i*m]=1;
            if(i%m==0) break;
        }
    }
    return p;
}
```
